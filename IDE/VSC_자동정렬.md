# VSC 자동정렬

날짜: 2021년 8월 25일

단축키 **cmd - k -f**

나는 먹히지 않아, extention Prettier을 사용한다.

저장하고 페이지를 껏다 키면 자동으로 정렬된다.

아래 단축키도 사용가능하다.

```jsx
1. CMD + Shift + P -> Format Document
OR
1. Select the text you want to Prettify
2. CMD + Shift + P -> Format Selection
```
